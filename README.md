# Olist E-commerce Analytics Case Study

## Цель проекта
Проанализировать продажи и поведение клиентов в интернет-магазине Olist, чтобы:
- выявить самые прибыльные категории товаров,  
- понять, какие пользователи делают повторные заказы,  
- дать рекомендации по увеличению выручки и удержанию клиентов.

## Описание проекта
Использованы данные маркетплейса Olist (Kaggle) для построения аналитики по продажам, поведению пользователей и ключевым метрикам продукта.  
Инструменты: Python, Pandas, Matplotlib, Seaborn, SQL.


# Анализ
### 1. Exploratory Data Analysis (EDA)
- Проверка пропусков и дубликатов  
- Анализ распределений заказов, выручки и категорий товаров  

### 2. Воронка продаж
- Анализ пути пользователя: первый заказ → повторные заказы → выручка  
- Рассчитаны коэффициенты повторных покупок по когортам  

### 3. Прибыльные категории
- Определены топовые категории по выручке и количеству заказов  
- Выявлены перспективные категории для cross-sell и upsell  

### 4. Поведение клиентов
- Разделение пользователей по активности и количеству повторных заказов  
- Анализ retention D1, D7, D30  

### 5. Гипотезы и рекомендации
- Персонализированные акции для новых клиентов увеличат Retention D7  
- Cross-sell между топовыми категориями повысит средний чек  
- Акцент на сегмент повторных покупателей увеличит LTV
